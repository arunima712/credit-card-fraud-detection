# Credit Card Fraud Detection

## Overview
This project detects fraudulent credit card transactions using Machine Learning algorithms.

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- SMOTE

## Models Used
- Logistic Regression
- Random Forest

## Features
- Data preprocessing
- Feature scaling
- Handling imbalanced data using SMOTE
- Model evaluation using Precision, Recall, F1-score and ROC-AUC

## Dataset
Kaggle Credit Card Fraud Detection Dataset
